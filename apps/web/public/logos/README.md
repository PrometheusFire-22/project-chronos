# Logo Assets Organization

## Directory Structure

### `final/`
Production-ready, approved logos for use in all marketing materials.
- **logo-ai-wordmark.svg** - Primary logo (transparent cutout, works on any background)
- **logo-wordmark-full.svg** - Full "automatonic ai" wordmark (finalized in Inkscape)

### `alternatives/`
Alternative approved variants for specific use cases:
- Kandinsky variant (for A/B testing, creative contexts)
- Minimal triangle (ultra-minimal contexts)
- Other graph-based variants

### `archive/`
Deprecated or superseded versions kept for reference only.
Do not use these in production.
